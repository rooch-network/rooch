Rooch Network 是比特币生态系统的原生应用层，很高兴地宣布我们与领先的比特币原生数据可用性（DA）层 Nubit 合作，准备通过最安全、可扩展的 DA 解决方案实现无限创新。此次战略合作标志着朝着提高比特币生态系统的安全性和可扩展性迈出了重要一步。

Nubit 是一个可扩展、经济高效的数据可用性层，由比特币为比特币社区提供保护。Nubit 能够扩展比特币的数据容量，为 Ordinals、Layer2 和价格预言机等应用程序提供支持，从而扩大比特币生态系统的范围和效率。Nubit 旨在充分继承比特币的安全性，包括经济安全性、抗篡改性、抗审查性，并在共识机制和数据可用性方面进行创新。借助 Nubit，比特币生态系统中的应用程序和用户可以享受由比特币经过验证的安全性保障的数据可用性层解决方案。Rooch Network 的核心是比特币生态系统的原生应用层，基于 Stackable L2 解决方案。它是比特币资产的首选启动板以及用户和开发人员的比特币应用程序的基础设施。Rooch Network 的使命是释放比特币经济的无限效用，致力于彻底改变区块链行业。

## Rooch Network 将集成 Nubit DA

Rooch Network 和 Nubit 合作的一个关键领域是将 Rooch Network 以资产为中心的基础设施与 Nubit 的先进 DA 解决方案集成。**具体来说，Rooch 将在 Rooch 节点内实现 Nubit DA 接口，使 Nubit DA 成为任何 Rooch 网络验证器的开箱即用选择。**

通过这种集成，开发人员将获得先进的数据可用性功能，使他们能够构建可扩展、安全且高效的区块链应用程序。此外，用户还可以在 Rooch Network 的预主网和主网上验证通过 Nubit DA 发布的交易。这确保了所有用户的透明和安全的环境，促进比特币生态系统的信任和可靠性。

## 合作开发下一代 CSV 资产标准

**另一方面，Rooch Network 正在与 Nubit 合作开发一个该行业应得的更好的 CSV 资产协议。**比特币生态系统是围绕我们在比特币（最安全的价值存储）上存储任意资产这一概念进行创新而启动的。但由于比特币区块链可编程性的限制，其应用场景匮乏。**我们相信，提出一个既是比特币原生又易于编程的新 CSV 资产标准是公平的。Rooch 和 Nubit 共同实施的索引器解决方案将支持这一新的资产标准，以提高其采用率。**

## 即将推出的 Rooch Network 与 Nubit 联合活动

**此外，Rooch Network 和 Nubit 正准备在 Nubit 测试网上发起联合活动。**该活动将展示合作伙伴关系带来的增强功能，允许用户与 Rooch Network 和 Nubit 进行交互，并查看提交给 Nubit 的 DA。

我们共同为比特币生态系统释放新的可能性，为开发人员提供无与伦比的机会来构建可扩展且安全的区块链应用程序。用户将受益于生态系统内交易和交互的透明度、安全性和效率的提高。我们对我们的合作伙伴关系将对比特币社区产生的积极影响感到兴奋，并期待共同推动创新！

## 关于 Rooch Network

Rooch Network 是比特币生态系统的本机应用层，基于 Stackable L2 解决方案，作为用户和开发人员的首选比特币资产启动板和比特币应用程序基础设施。

## About Nubit

Nubit 由 UCSB 教授 Yu Feng 创立，利用比特币的经济安全性来创建高度安全且可扩展的数据可用性层。凭借大量资金，包括由 Polychain Capital 领投的大型种子轮融资，以及零知识技术的整合，Nubit 推进了区块链可扩展性，以支持 AI、SocialFi 和 GameFi 等领域的无限应用。Nubit 的创新解决方案为各个领域开启了新的可能性，确保无与伦比的信任和可靠性。作为比特币第 2 季的领导者，Nubit 已将其技术与主要行业参与者集成，并将支持扩展到领先的第 2 层框架。
