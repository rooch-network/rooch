我们很高兴宣布，Rooch Network 与 **World3** 携手合作，共同推出激动人心的 **双挖计划**！World3 是一个去中心化平台，AI代理与人类共同创造、创新与进化。通过参与此计划，您将能够在 **Grow Bitcoin** 平台上同时获得来自两个前沿生态系统的奖励，助力比特币、虚拟世界和AI生态的共同成长。通过参与此计划，您可以同时获得来自 World3 和 Rooch Network 的丰厚奖励，而且完全免费！

### 如何参与：

1. 访问Rooch Network的[Grow Bitcoin平台](https://grow.rooch.network/)。
2. 通过完成平台任务赚取 $GROW。
3. 进入World3页面，使用您的 $GROW 代币为World3投票。
4. 活动结束时间：**2月20日 4PM（UTC+8）**

### 活动亮点：

- **双重奖励机制**：赢取来自 **World3** 的 Lumens 和 **Rooch** 平台的独家奖励。
    - 花费 $GROW 进行投票并攀升排行榜——前 2,000 名参与者将共享 **540,000 Lumens**。
    - Lumens 可以在 World3 生态中使用，或保存用于未来的 **World3 代币空投**。
    - 所有参与者还将获得 **$GROW** 和 **BitXP**，这些奖励将与未来的Rooch空投活动挂钩。

- 排名不在前2,000的用户，将通过抽奖分享 **60,000 Lumens（10%）**。

### 活动规则：

1. **如何参与**
    
    用户需在活动页面完成投票任务，才能获得参与资格。
    
2. **奖励申领流程**
    
    活动结束后，用户需在 **2月21日 - 2月25日** 提交 **World3 账号信息**，用于奖励发放。未在规定时间内填写账号信息的用户将视为放弃奖励。
    
3. **发放奖励**
    
    奖励将在 **2月25日后的一周内** 发放。
