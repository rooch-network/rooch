# Rooch AI agent 校园创意挑战赛：AaaS 脑洞征集大赛

Agent-as-a-Service (AaaS) 是一种创新的服务模式，通过自主智能代理（AI agent）为用户提供按需服务。它可以像贴心助手一样，自动完成复杂任务——无论是优化 Web2 平台的社交体验，还是推动 Web3 去中心化应用的智能化发展，AaaS 都展现出无限可能。

Rooch Network 相信，大学生群体充满新奇想法和创造力，是推动智能未来发展的生力军。我们希望通过此次大赛，从你们的“新脑子”中挖掘 AaaS 的创新应用，连接 Web2 与 Web3 的场景，同时正确引导你们进入 Web3 世界，探索去中心化技术的魅力！

### 活动详情

- **面向人群**：全球大学生（本科生及研究生）
- **形式**：线上脑洞征集，提交创意提案（文字形式，500-2000 字）
- **时间**：2025年4月14日 - 4月30日
    - 报名与创意提交：4月14日 - 4月24日
    - 评审与颁奖：4月25日 - 4月30日
- **要求**：
    - **参赛者需提出 AaaS 在 Web2、Web3 或两者结合领域的创新脑洞，提交一份详细的创意提案，内容包括应用场景的描述、所解决的具体问题、创新点以及潜在价值（无需 demo 或技术实现）。**
    - 提案必须为参赛者的原创观点，展现独特的思考和创意，不得抄袭他人作品或引用未经授权的内容。

### 选题方向参考

1. Web3 社交与身份管理：如去中心化社交匹配、跨平台身份管理、社交信用评估。
2. Web3 金融与交易：如 DeFi 收益优化、跨链交易助手、智能风险预警。
3. Web3 内容创作与版权保护：如 NFT 内容创作助手、版权追踪、去中心化众筹。
4. Web3 游戏与娱乐：如 GameFi 策略助手、元宇宙活动策划、NFT 交易优化。
5. Web2-Web3 跨界场景：如跨界支付、内容分发、统一身份管理。
6. Web2 社交与生活服务：如智能内容推荐、生活服务优化助手。
7. 开放脑洞：任何与 AaaS 相关的创意都可以。
