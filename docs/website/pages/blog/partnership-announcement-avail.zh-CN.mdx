**比特币的应用层 Rooch Network 很高兴地宣布与 Avail 建立战略合作伙伴关系，Avail 是一个模块化区块链解决方案，旨在统一 Web3 并优化数据可用性（DA），以实现高度可扩展和可定制的汇总（rollup）。**通过将 Rooch Network 以资产为中心的基础设施与 Avail 最先进的数据可用性（DA）层相集成，我们正在为区块链可扩展性、效率和互操作性制定新标准。

**Rooch Network 的核心是作为比特币的应用层，提供以资产为中心的基础设施，为所有比特币资产提供索引、交易和应用场景。**Rooch Network 的使命是释放比特币经济的无限可能，致力于彻底改变我们所知的区块链行业。

**Avail 产品的核心是 Avail DA，它是一个基础 DA 层，它实现了尖端技术，包括 KZG 承诺和数据可用性采样（DAS）。**Avail DA 使轻客户端能够通过点对点网络采样轻松验证数据，从而彻底改变了区块链数据的可用性，从而提高了开发人员的安全性和效率。Avail 高效且可扩展的 DA 解决方案与 alt-VM（尤其是超高性能 MoveVM）完美搭配，使 Rooch Network 上的开发人员能够自信地部署生产就绪代码，而不会面临代价高昂的安全漏洞的迫在眉睫的威胁。

**通过将 Avail 的 DA 集成到 Rooch Network 的基础设施中，开发人员将获得高级数据可用性功能，使他们能够构建可扩展、安全且高效的区块链应用程序。**Rooch Network 和 Avail 之间的合作标志着 Web3 和去中心化应用程序发展的一个重要里程碑。凭借 Avail 对强大且可扩展的区块链基础设施的承诺以及 Rooch Network 对增强智能合约安全性的关注，开发人员已准备好创建独特且有影响力的在线体验，任何有互联网连接的人都可以访问。

最后但并非最不重要的一点是，Rooch Network 将很快推出其预主网，而 Avail 的最终测试网络 Turing 刚刚上线，为即将推出的 Avail DA 主网铺平了道路。请继续关注 Rooch Network 和 Avail 的进一步更新，不断突破区块链技术的界限并推动去中心化解决方案的采用。

## 关于 Avail

Avail 由 Polygon 前联合创始人 Anurag Arjun 领导，正在构建一个统一层来大规模解决 Rollup 碎片问题。Avail 从使用 Avail DA 解决区块链可扩展性的首要原则出发解决了这个问题，Avail DA 是一个基础 DA 层，它实现了为以太坊 danksharding 路线图规划的相同技术，包括 KZG 承诺和数据可用性采样（DAS）。Avail Nexus 利用 Avail 可扩展 DA 层上的证明聚合，通过无需许可的互操作性解决了日益增长的碎片问题。然后，通过 Avail Fusion 通过多资产质押来增强 Avail 的安全性。立即在 `availproject.org` 开始您的统一之旅。

## 关于 Rooch Network

Rooch Network 是比特币的应用层，是一个以资产为中心的基础设施，为所有比特币资产提供索引、交易和应用场景。Rooch Network 采用以资产为中心的方式，将资产作为其基础设施设计的中心，彻底改变了去中心化的实现方式。Rooch Network 致力于释放比特币经济的无限可能。
