Rooch Network 很高兴与 PAN Network 达成合作，**PAN Network** 是一个开创性的项目，正在通过提升闪电网络的可扩展性、安全性和易用性来改变比特币支付。我们将共同在 Grow Bitcoin 平台上推出 双挖计划，为参与者提供赚取 **$GROW**、**$xPAN** 积分以及解锁独家奖励的机会。无论你是矿工、NFT 收藏者，还是比特币爱好者，这都是一次参与在两个前沿生态中的机会，同时赚取宝贵奖励！

### 如何参与：

1. 请访问 [**Rooch Network 的 Grow Bitcoin 平台**](https://grow.rooch.network/)来开始。
2. 通过参与 Grow Bitcoin 平台上的活动来赚取 **$GROW**。
3. 转到 **PAN Network** 页面，并使用您的 **$GROW** 代币为 PAN Network 投票。
4. 该活动于**2月24日下午6点结束（UTC+8）**。

### 活动亮点：

- **双挖奖励**：从 Pan Network 和 Rooch 独家奖励中赚取 **$xPAN** 积分。
- **$xPAN** 积分是从 Pan Network 中解锁有价值的奖励的基石。
- 在代币生成活动（TGE）之后，**$xPAN** 积分将被以 **$PAN** 代币的形式进行空投，从而大大提高其价值。
- 顶级参与者将获得慷慨的 **$xPAN** 奖励。攀登排行榜，并在顶级矿工中确保您的位置！

### 奖励分布：

**排行榜奖励**：

| **排名**       | **独立的奖励** |
|----------------|----------------|
| **1 - 100**    | 80 xPAN each   |
| **101 - 500**  | 40 xPAN each   |
| **501 - 1000** | 20 xPAN each   |

### 活动规则：

1. **如何参与**：
    - 在活动页面上完成投票任务，以获得参与的资格。
2. **奖励分布**：
    - 活动结束后，将向 Rooch 社区提供有关奖励分配的详细信息。

### 为什么要加入双挖计划？

这不仅是另一个奖励计划，而且是一个令人兴奋的机会，成为比特币和比特币支付十字路口不断发展的生态系统的一部分。通过参加，您不仅会获得独家奖励，而且还有助于扩展 Rooch Network 和 PAN Network，在去中心化的财务和比特币应用的未来中发挥着关键作用。

不要错过加入两个创新平台，挖矿奖励的机会，并对快速增长的比特币生态系统产生有意义的影响。准备赚钱了吗？让我们开始挖矿吧！
