Rooch v0.3 发布，接入比特币网络，实现铭文铭刻，启动测试网（Testnet）。这是 Rooch 第三个里程碑（[Rooting](/learn/miscellaneous/roadmap)）的一部分。

## Rooch v0.3 的关键特性

这个版本主要提供了以下主要特性：

### DA

Rooch 目前实现了 DA 开放网络框架，后续可以支持以下场景：

* 可接入像 Celestia 这样的 DA Chain
* 也可以接入云存储作为快速存储介质
* 还可以为测试或缓存服务开启本地存储
* 协同作业，多 DA 后端可在开放网络内协同工作，在满足不同需求的同时实现高可靠性。

### Bitcoin 支持

* 接入 Bitcoin 测试网络，将 UniSat 作为默认钱包，详细使用方法请参考安装 UniSat。
* Rooch 实现 Bitcoin 中继网络，为成为 Bitcoin 网络执行层提供基本支持。
* 实现交易类型、交易进度、地址、余额、铭文、UTXO 等解析。

### 对象存储

这一版本对对象存储做了重大重构，详细使用请参考[对象](../learn/core-concepts/objects/object)，为开发者提供更好的开发体验。

### 铭文

* 目前实现了铭文铭刻、展示、查询等基础功能，详细操作请参考铭文。

### Rooch Dashboard

* 提供 Bitcoin 网络交互功能
* 集成 UniSat 钱包，打开 Dashboard 将自动检测、验证、授权、登陆
* 提供铭文查询功能
* 访问地址：[https://dashboard.rooch.network/](https://dashboard.rooch.network/)。
* 使用说明请参看：Rooch Dashboard 使用说明。

### JSON RPC 以及 SDK

* 实现 Bitcoin 相关 RPC
  * 提供 UTXO 查询接口：`btc_queryUTXOs`
  * 提供铭文查询接口：`btc_queryInscriptions`
* SDK 添加 Bitcoin 相关操作

Rooch 当前支持的接口请参看：[Rooch JSON RPC](../build/reference/rpc/json-rpc)。

### 测试网

测试网络信息：

1. 名称: test
2. ChainID: 2
3. RPC: https://test-seed.rooch.network/

如何使用测试网，请参考文档：[连接测试网络](../build/getting-started/connect-to-rooch/connect-testnet)

## 下一步的目标

1. 提供永久激励测试网络，让开发者可以在上面部署应用，让用户试用。
2. 运行 RoochBTC 先行网，实现比特币主网资产的索引。关于这个目标的详细描述请参阅 [RoochBTC - 用链上索引器实现比特币二层](https://rooch.network/zh-CN/blog/rooch-btc)。

## 总结

Rooch 的里程碑遵循应用场景驱动的原则，以开发者和用户的使用体验为中心，尝试拓展 Web3 应用的边界，探索 Web3 大规模采用落地的可能。想了解并试用 Rooch，请参看文档：[快速开始](../build/tutorial/counter)。
