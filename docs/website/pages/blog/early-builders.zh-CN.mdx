自成立以来，Rooch Network 已与 BTC 和 Move 生态系统中众多受人尊敬的团队建立了牢固的联系。**为了进一步加强这些合作伙伴关系并欢迎新的合作，我们很高兴推出早期构建者计划。该计划邀请开发人员在 Rooch 网络上构建应用程序，并作为我们即将推出的资助计划的先驱。该计划的参与者将有机会分享 1,000,000 $ROOCH 的奖池**。

## 怎么进行的？

申请正式开放。主要关注比特币生态系统的应用。欢迎所有 DeFi、基础设施、工具、游戏、元宇宙、NFT、治理等领域的优质项目。Rooch Network 旨在构建比特币生态系统的应用层。基于这一使命，以下是考虑提案的基本要求：

- 该提案的主题应该围绕以比特币为中心的应用程序。
- 支持比特币生态资产开发。
- 展示产品与市场契合度的用户体验插图。

### 第 1 步

- 请查看 Rooch Network 的关键资源并填写基本信息表：*[Rooch Network 早期构建者计划申请表](https://forms.gle/WTfAcdPrSE4BQvnh8)*。您的申请越具体和清晰，我们的团队就越能了解您的情况项目建议书。
- 加入我们在 *[Discord](https://discord.com/invite/rooch)* 上的 Early Builder 频道，与 Rooch 团队进行直接对话。
- 在收到申请后三天内，Rooch Network 将通知申请人其提案是否已被接受。Rooch 团队将根据技术实力、规格细节、团队经验以及实用性来评判申请。

### 第 2 步

- 一旦您收到您的提案已被接受的通知，您就可以开始部署演示。
- 部署完成后，需要编写开发文档。
- 我们将在收到您的演示和文档后一周内通知您结果。

### 第 3 步

- 优秀的演示将被选入我们的资助计划，并将获得额外的资金和支持。

## 奖励详情

- 每个提交提案的参与者将收到 1,000 $ROOCH。
- 每个部署演示并编写文档的参与者将获得 50,000 $ROOCH。
- 优秀的文档将出现在 Rooch 的博客上。优秀的开发者将有机会加入 Rooch 团队。
- 杰出的演示将被选入 Rooch 资助计划，并将获得额外的资金和支持。
- 奖品将在 $ROOCH TGE（代币生成活动）期间分发到您的帐户。
