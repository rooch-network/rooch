我们很高兴宣布，Rooch Network 与 UXLINK达成战略合作，在 Grow Bitcoin 平台上推**双挖计划**！此次合作让参与者能够在活动期间同时获得来自 UXLINK 和 Rooch Network 的丰厚奖励。不要错过这个无需成本即可最大化收益的绝佳机会！

![](/blog/partnership/uxlink/uxlink-zh.png)

### 参与方式：

1. 访问 Rooch Network 的 [Grow Bitcoin 平台](https://grow.rooch.network/)。
2. 通过参与 Grow Bitcoin 平台赚取 $GROW。
3. 前往平台的 UXLINK 页面，使用 $GROW 代币为 UXLINK 投票。
4. 活动截止时间：2025 年 1 月 5 日晚上 8 点（UTC+8）。

### 活动亮点：

- **双重奖励机制**：同时获得 **UXLINK 积分** 和 **Rooch 平台专属奖励**。
    - **用于投票的$GROW 越多，您的排名越高。投票数在前 1,000 名的地址** 将每人获得 **100 UXUY**，共计 **100,000 UXUY**。
    - **UXUY** 是 UXLINK 生态系统内的奖励积分，用于平台上的工作量证明（POW）活动。这些积分可用于参与未来的 **$UXLINK 空投**。
    - 所有参与者还将从 Rooch Network 获得 **$GROW** 和 **BitXP** 奖励，这些奖励与未来的 Rooch 空投挂钩。

### 为什么参与？

Rooch x UXLINK的双挖活动让您同时享受来自两个创新 Web3 生态系统的双重奖励，提升您的资产的价值。参与双挖计划，不仅能轻松获取丰厚奖励，还能助力 UXLINK 和 Rooch Network 的生态发展， 立即加入双挖计划，与 Rooch 和 UXLINK 一起携手，共同开创 Web3 新篇章吧！
