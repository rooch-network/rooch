我们很高兴宣布，Rooch Network 与 Yescoin 达成战略合作！Yescoin 是 Telegram 上最大、增长最快的mini-app之一，此次合作将在 Grow Bitcoin 平台上推出 **双挖计划**。通过参与此计划，您可以同时获得来自 Yescoin 和 Rooch Network 的丰厚奖励，而且完全免费！

### 如何参与：

1. 访问 Rooch Network 的 [Grow Bitcoin 平台](https://grow.rooch.network/)。
2. 通过参与 Grow Bitcoin 平台赚取 $GROW。
3. 前往平台上的 Yescoin 页面，用您的 $GROW 代币为 Yescoin 投票。
4. 截止时间：1月27日下午8点（UTC+8）。

### 活动亮点：

- **双重奖励机制**：**获得 YESCOIN 积分** 和 **Rooch 平台独家奖励**。
    - 使用 $GROW 投票的数量越多，排名越高！前 2,000 名地址将瓜分 77,250,000 YESCOIN 积分。
    - YESCOIN 是 Yescoin 生态系统中的激励积分，通过平台上的工作量证明（POW）活动获得。这些积分可用于参与未来的 Yescoin **空投**。
    - 所有参与者还将从 Rooch Network 获得 **$GROW** 和 **BitXP** 奖励，这些奖励与 Rooch 的未来空投密切相关。
