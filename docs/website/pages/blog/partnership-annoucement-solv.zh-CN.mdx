**我们很高兴地宣布 Rooch Network 与 Solv Protocol 之间建立战略合作伙伴关系，Solv Protocol 是由创新性质押抽象层 (SAL) 提供支持的比特币质押领域的领导者。此次合作不仅增强了比特币的实用性，还为 SolvBTC 持有者带来了新的、有价值的收益机会，突破了去中心化金融的界限。**

Solv Protocol 已牢牢确立了自己作为比特币质押领跑者的地位，锁定总价值 (TVL) 超过 24 亿美元，质押超过 25,000 个 BTC。通过其旗舰产品 SolvBTC 和面向所有人的比特币储备，Solv Protocol 旨在释放超过 1 万亿美元比特币资产的全部潜力。借助 SolvBTC Liquid Stake 代币（LST），比特币持有者可以在保持流动性的同时获得多种收益产生机会，从而无缝参与 DeFi 生态系统。 Solv 正在为机构和传统基金自信地进入比特币金融 (BTCFi) 世界铺平道路，缩小加密货币与主流金融之间的差距。

作为以 MoveVM 和比特币质押为特色的领先比特币应用层，Rooch Network 将原生比特币应用程序和收益生成带入完全非托管环境。通过与 Solv Protocol 合作，Rooch 为比特币持有者开辟了新的可能性，在整个生态系统中提供创新的用例、更大的流动性和更高的收益机会。

## Rooch Network 解锁 SolvBTC 的实用性和收益

**Rooch Network 将通过为 SolvBTC 持有者提供创新用例和场景，为 Solv 生态系统做出贡献。**具体来说，SolvBTC将直接集成到Rooch生态系统内的DeFi、GameFi和其他应用程序中。 SolvBTC 持有者将获得 Rooch Network 上独特的收益机制和动态奖励生态系统。此次合作预计将促进基于比特币的金融（BTCFi），增强比特币质押体验，并为比特币持有者引入新的原生比特币应用程序。

## 联合探索比特币第一层原生质押协议

除了将 SolvBTC 集成到 Rooch 的生态系统之外，**Rooch Network 和 Solv Protocol 还将共同探索比特币一层的原生质押协议，为比特币质押的大规模采用铺平道路。**

Solv Protocol 新推出的质押抽象层（SAL）是一套智能合约，旨在简化和标准化跨链的比特币质押。它促进比特币主网和 EVM 兼容链之间的无缝交互，同时与 LST 发行者、质押协议和验证器集成。在此基础上，Rooch Network 和 Solv Protocol 将探索 SAL 与 MoveVM 的兼容性，并为比特币第 1 层开发原生质押协议。该计划旨在提高比特币质押领域内的互操作性，使其更易于所有用户使用。

## 接下来有什么？

在未来几周内，Rooch Network 将推出其预主网，并为比特币生态系统提供真正的收益和效用。与 Solv Protocol 集成是我们致力于扩大 BTC 在各种区块链生态系统中的效用和收益的承诺的一部分。我们邀请 BTC 持有者参与这一创新解决方案，分享更加去中心化的未来的经济安全和回报。如需更多更新和详细说明，请关注我们的频道。
