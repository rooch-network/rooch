# 关于 Rooch

## 愿景

Rooch 项目的使命是打造一个能够承载 Web3 原生应用的基础设施，并彻底的改变我们当前互联网应用的中心化格局。

我们的想法是做出一个好的基础设施产品，让这个产品为自己代言。并且这个产品能承载更多更好的 Web3 产品来做到同样的事情。

运用社区的力量，我们将引领 Web3 应用的爆发并迁移下一个十亿级别的用户。

## Rooch DAO

Rooch 是一个开源的项目，欢迎任何人参与贡献。为了更好的让社区参与，我们将在近期发布 Rooch DAO 的规章细则。

如果你有任何想法或者建议，欢迎加入我们的 Discord 并和我们交流。
