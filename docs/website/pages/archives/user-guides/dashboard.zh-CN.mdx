# Rooch 仪表盘介绍

Rooch 的仪表盘为用户提供一站式服务，包括账户总览、交易查询、钱包管理、交易发送、认证管理等众多功能。

![仪表盘首页](/docs/dashboard-home-page.png)

Rooch 仪表盘 URL：

- 连接测试网：[https://dashboard.rooch.network/](https://dashboard.rooch.network/)
- 连接开发网：[https://dev-dashboard.rooch.network/index.html](https://dev-dashboard.rooch.network/index.html)

> 注意：当前测试网仅支持比特币网络，开发网仅支持以太坊网络。

## 与比特币网络交互

### 登陆

打开 [Rooch 官网](https://rooch.network/)，点击导航栏上的 `Dashboard` 按扭。首次登陆时，默认会自动弹出 UniSat 钱包的连接提示：

![](/docs/dashboard-btc-login.png)

点击 `Connect` 按扭，同意登陆，就可以进入 Dashboard 页面了。

![](/docs/dashboard-btc.png)

### 导航栏介绍

仪表盘页面中顶部为导航栏，包括搜索框、明暗主题切换按钮、语言切换按钮、网络切换按钮，已连接的钱包地址。

### 侧边栏介绍

当前可用的功能包括：状态查询、交易查询，铭文资产。

状态查询，点击 `List` 方法，可以列出当前 `UTXO` 对象。在列表中会展示对象类型、对象 ID、对象所有者这三个属性。

交易查询，点击 `Transaction` 按扭，可以查看交易信息，默认列出最近 10 条信息。

![](/docs/dashboard-btc-transaction.png)

铭文资产，这里展示铭文对象的信息，每个铭文对象包含：铭刻时间、对象 ID、对象类型、对应比特币链上的交易哈希、元数据等这些属性。

![](/docs/dashboard-inscription.png)

> 注：当前还在快速开发中，后续功能会陆续推出！

## 与以太坊网络交互

### 登陆

打开 Rooch 仪表盘地址，可以看到如下的登陆界面，默认选择 `MetaMask` 钱包，点击 `CONNECT` 按钮，即可登陆。

![仪表盘登陆页面](/docs/dashboard-login.png)

#### Session key

首次登陆认证会提示用户认证 Session key，Session key 有过期时间限制，如果过期需要再次验证：

![](/docs/dashboard-session-key.png)

### 导航栏介绍

仪表盘页面中顶部为导航栏，包括搜索框、明暗主题切换按钮、语言切换按钮、网络切换按钮，已连接的钱包地址。

搜索框中可以搜索交易、地址、事件等。

### 侧边栏介绍

侧边栏是 Rooch 仪表盘各种功能的快速导航，点击相应的控件可以快速跳转到相应的页面。
