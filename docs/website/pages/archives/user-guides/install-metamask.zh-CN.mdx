# MetaMask

MetaMask 是一款基于浏览器扩展的钱包，它运行在 Web 浏览器（Chrome、Firefox 或 Edge）中。MetaMask 最初是为以太坊区块链开发的一款浏览器钱包，现在只要区块链网络实现其相关的接口就能够跟相应的区块链网路进行交互。Rooch 网络目前支持使用 MetaMask 进行交互和管理加密资产。

## 安装

打开 Google Chrome，访问 [Chrome Web Store](https://chrome.google.com/webstore/category/extensions)：

![](/docs/install-metamask-search.png)

点击 `Add to Chrome` 按钮：

![](/docs/install-metamask-add.png)

如果有弹出提示，请选择 `Add extension` 按钮：

![](/docs/install-metamask-confirm-add.png)

- 1：选择合适的界面语言
- 2：勾选使用条款
- 3：创建一个新钱包
- Or：你之前已经有 MetaMask 钱包了，可以选择导入已有的私钥来恢复钱包

![](/docs/install-metamask-create.png)

默认选择 `I agree`：

![](/docs/install-metamask-agree.png)

输入本机上解锁 MetaMask 钱包的密码，勾选使用条款，并点击 `Create a new wallet` 按钮：

> 注意：这个密码跟私钥是不同的！

![](/docs/install-metamask-login-passwd.png)

默认选择 `Secure my wallet`：

![](/docs/install-metamask-login-confirm.png)

点击 `Reveal Secret Recovery Phrase` 按钮，显示私钥助记词：

![](/docs/install-metamask-show-phrase.png)

找一张纸把助记词记下来，点击 `Next`：

![](/docs/install-metamask-write-down.png)

填写欠缺的助记词，确认抄写的助记词是正确的，并点击 `confirm` 按钮：

![](/docs/install-metamask-write-down-confirm.png)

当你看到这个界面，就表示你的以太坊钱包已经创建完成了，开始你的区块链之旅吧！

![](/docs/install-metamask-got-it.png)

这是 MetaMask 钱包主界面：

![](/docs/install-metamask-main.png)

为了方便访问，我们可以将 MetaMask 的图标固定到浏览器的导航栏。

点击浏览器的右上角的插件图标：

![](/docs/install-metamask-addon.png)

点击这个钉子图标就可以切换是否固定 MetaMask 到导航栏了：

![](/docs/install-metamask-fixed.png)

至此，你已经完成了 MetaMask 安装了，接下来将介绍如何使用 MetaMask 跟 Rooch 网络交互。
