Rooch（/ruʧ/）的命名是 Root 和 Branch 的结合，象征着我们从种子中长出根和枝，并将种子传播到森林生态系统中所获得的灵感。我们相信比特币是我们的种子，构建比特币生态系统是实现 Web3 愿景的途径。

在本文档中，您将了解 Rooch 如何设计为可验证的应用程序容器并充当比特币的应用程序层。如果您有兴趣，还请浏览我们的博客文章以进行更多技术讨论。
