# Rooch 的内置库

Rooch 当前内置了四个标准库，分别是 `MoveStdlib`、`MoveosStdlib`、`RoochFramework` 和 `BitcoinMove`。

四个库在 Move 中的地址分别是：

- `MoveStdlib`：`0x1`
- `MoveosStdlib`：`0x2`
- `RoochFramework`：`0x3`
- `BitcoinMove`：`0x4`
