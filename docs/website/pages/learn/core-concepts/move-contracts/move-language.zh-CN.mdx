# Move 语言

import Image from "next/image";

<Image src="/docs/move-logo.png" alt="Move Logo" width={300} height={300} />

Move 是由 Diem 最初设计和开发的开源智能合约编程语言。

`Move` 的命名来自于 [C++ 中的移动语义](https://stackoverflow.com/questions/3106110/what-is-move-semantics)，它被用于将资源从一个对象移动到另一个对象而不进行复制。这种设计目的完全符合智能合约语言的上下文，并使其成为实现加密资产的灵活性和安全性的完美选择。

> 了解为什么 Rooch 在[从 Web2.5 到 Web3](/blog/from-web2.5-to-web3#smart-contract-programming-language) 和[为什么是 Move 之编程语言的生态构建](/blog/why-move-ecosystem)中采用 Move 语言。
