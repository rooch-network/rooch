# 存储

Rooch 通过存储抽象（**StorageAbstraction**），让开发者可以在智能合约中更灵活地定义应用的状态存储结构，而不局限于平台提供的标准化方案，更适合全链应用。具有以下特点：

1. 提供全局的 `Object` 存储，也提供用户存储空间（**AccountStorage**)，方便开发者根据业务场景进行存储划分。
2. 开发者可以通过[私有泛型](../../../build/rooch-framework/private-generics)来定义存储结构的访问和扩展权限。
3. 为未来应用专用的存储空间和应用专用的运行容器提供基础。
