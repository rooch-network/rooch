# Rooch Framework Document

Rooch currently includes three built-in Move standard libraries, namely `MoveStdlib`, `MoveosStdlib`, and `RoochFramework`. 
The addresses of the three libraries in Move are as follows:

* `MoveStdlib` : `0x1`
* `MoveosStdlib` : `0x2`
* `RoochFramework` : `0x3`
