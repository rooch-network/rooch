# Rooch Framework 文档

Rooch 当前内置了三个 Move 标准库，分别是 `MoveStdlib`、`MoveosStdlib` 和 `RoochFramework`。
三个库在 Move 中的地址分别是：

* `MoveStdlib` : `0x1`
* `MoveosStdlib` : `0x2`
* `RoochFramework` : `0x3`
