## Rooch 将哪些应用程序定位为基础设施？Rooch 如何吸引用户和开发者？

包括我们在内的所有模块化区块链基础设施，我们都在努力推动区块链技术的边界，因此区块链应用程序可以在限制更少、功能更多的环境中发展。我相信，我们可以用“更少的限制和更多的功能”来做什么的问题是所有 Layer2 或模块化基础设施的共同问题。

一方面，我们还没有在任何 Layer2 或模块化区块链上看到任何杀手级应用。另一方面，目前市场上活跃的应用，比如 DeFi 和 NFT，他们的设计和实现都比较简单，也大多在 Layer1 上直接构建。所以这个问题的答案本质上是一个愿景问题——你相信会有更好的基础设施支持更复杂的去中心化应用程序吗？

我们的答案是肯定的，我们相信模块化基础设施将支持杀手级 Web3 应用程序，这是我们以前从未见过的。但是，我们还不能展示该应用程序，因为它需要整个行业的努力才能达到这一点。我们只是根据我们的技术直觉得到了这个答案。但我们很清楚，在 Rooch，我们不仅需要花精力构建基础设施，还需要探索应用程序。我们有一些想法，但只是在非常早期的阶段，而且很难描述。我们可能会在今年晚些时候进行一些具体的演示。
