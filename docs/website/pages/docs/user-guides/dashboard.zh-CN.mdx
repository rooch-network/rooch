# Rooch 仪表盘介绍

Rooch 的仪表盘为用户提供一站式服务，包括账户总览、交易查询、钱包管理、交易发送、认证管理等众多功能。

![仪表盘首页](/docs/dashboard-home-page.png)

Rooch 仪表盘的登陆地址为：[https://dashboard.rooch.network/login/](https://dashboard.rooch.network/login/)

## 登陆

打开 Rooch 仪表盘地址，可以看到如下的登陆界面，默认选择 `MetaMask` 钱包，点击 `CONNECT` 按钮，即可登陆。

![仪表盘登陆页面](/docs/dashboard-login.png)

## 导航栏介绍

仪表盘页面中顶部为导航栏，包括搜索框、明暗主题切换按钮、语言切换按钮、网络切换按钮，已连接的钱包地址。

搜索框中可以搜索交易、地址、事件等。

## 侧边栏介绍

侧边栏是 Rooch 仪表盘各种功能的快速导航，点击相应的控件可以快速跳转到相应的页面。
