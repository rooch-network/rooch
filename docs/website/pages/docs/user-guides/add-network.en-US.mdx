# Add Rooch network to MetaMask

## Addition steps

1. Open MetaMask and select the `network control` button in the upper left corner:

![MetaMask Network Entry](/docs/metamask-network-entry-en.png)

2. Click `Add network` below:

![](/docs/metamask-add-network-en.png)

3. Click `Add a network manually` at the bottom of the page to add a network:

![](/docs/metamask-manual-add-en.png)

4. On the page for manually adding a network, fill in the four required information: `Network name`, `New RPC URL`, `Chain ID`, and `Currency symbol`.

## Dev network

- Network name: `Rooch Devnet` (customizable)
- New PRC URL: `https://dev-seed.rooch.network`
- Chain ID: `20230103`
- Currency symbol: `RGC` (meaning 'Rooch Gas Coin')

Add Rooch's Dev network as follows and save:

![](/docs/metamask-devnet-en.png)
