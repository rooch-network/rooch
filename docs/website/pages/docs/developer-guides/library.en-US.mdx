# Rooch's built-in library

Rooch currently has three built-in standard libraries, namely `MoveStdlib`, `MoveosStdlib` and `RoochFramework`.

The addresses of the three libraries in Move are:

- `MoveStdlib`: `0x1`
- `MoveosStdlib`: `0x2`
- `RoochFramework`: `0x3`
